# Product

## Register

product

## Platform

web

## Users

Security analysts investigating endpoint activity in a focused SOC workflow. They need to triage incidents quickly, understand deterministic evidence, inspect raw telemetry safely, and distinguish compromise from authorized activity.

## Product Purpose

Nolen turns Linux endpoint telemetry into explainable incidents. Success means an analyst can move from an active incident to its entities, timeline, ATT&CK techniques, and raw supporting evidence without losing provenance or exposing secrets.

## Positioning

Every incident is an understandable, deterministic story backed by inspectable evidence.

## Brand Personality

Vigilant, lucid, and grounded. The interface should create calm operational focus rather than cinematic threat drama.

## Anti-references

Avoid neon “hacker” aesthetics, decorative terminal chrome, opaque risk scores, generic card walls, and visual severity cues that rely on color alone.

## Design Principles

- Evidence before decoration.
- Preserve context while analysts move from overview to detail.
- Make severity, confidence, and system state unambiguous without manufacturing certainty.
- Treat hostile and sensitive telemetry as inert data everywhere.
- Prefer familiar keyboard-accessible controls under operational pressure.

## Accessibility & Inclusion

Meet WCAG 2.1 AA, preserve visible keyboard focus, support reduced motion, provide non-color state labels, and keep dense evidence legible across desktop and narrow web layouts.
